# Информация и отчёт

В данной папке размещены материалы, не предназначенные для участников CTF.

Содержимое папки используется для:
- отчёта по выполненной работе;
- описания логики заданий;
- хранения решений и пояснений к каждому этапу;

Файлы из данной директории **не участвуют в работе сайта** и **не влияют на прохождение CTF**.
